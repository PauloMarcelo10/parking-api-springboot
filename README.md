#  Parking API - Spring Boot

##  About the project
REST API for parking management, allowing user registration, clients, parking spots and vehicle check-in/check-out.

---

##  Technologies
- Java
- Spring Boot
- Spring Security
- JWT
- SpringDoc OpenAPI (Swagger)
- JPA / Hibernate
- MySQL
- Maven
- Postman (API testing)

---

##  Features
- User authentication with JWT
- User registration
- Client management
- Parking spot management
- Vehicle check-in
- Vehicle check-out
- Exception handling

##  API Documentation
After running the project, the API documentation will be available via Swagger (SpringDoc):

http://localhost:8080/swagger-ui.html
